# Corona Navigator

## Run

### Run Locally
#### Install Python Modules
Run
```pip install -r requirements.txt```
#### Start Server
Run
```python app.py```
#### Start Frontend (no Server needed locally)
Run <br>
```yarn install```<br>

```yarn serve```
#### Start Local Database
Run <br>
```export MONGO_URI="mongodb://localhost:27017/serviceDB"```<br>

```make database```

### Build Frontend
```yarn build```

### Deploy
* push to master
